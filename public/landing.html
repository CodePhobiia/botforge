FAQ
Answers to the most common BotForge questions.
Do you store my Discord token or API key?

Keys are encrypted in transit and only used to run your bot. You can rotate or revoke them at any time.

Can I use multiple AI providers?

Yes. Each bot can be configured with its own provider and model.

What happens if my bot goes offline?

The dashboard shows uptime indicators so you can restart quickly. Our team plan includes priority support.

Is there a free plan?

Yes, you can run one bot with up to 50 messages per day for free.
